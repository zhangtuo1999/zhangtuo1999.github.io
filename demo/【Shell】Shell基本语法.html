1 Shell 概述

Shell是一个命令行解释器，用于接收应用程序/用户的命名，然后调用操作系统的内核。

查看当前可用的Shell解释器：

cat /etc/shells

查看当前使用的Shell解释器：

echo $SHELL

2 Hello World

2.0 创建一个脚本

vim helloworld.sh

2.1 指定解析器

Shell 脚本的开头必须指定 Shell 脚本的解析器。格式为：

#!/bin/bash

或者

#!/bin/sh

2.2 添加命令

添加一行输出”Hello World”命令后保存退出：

#!/bin/bash
echo "Hello World"

2.3 执行命令

sh ./helloworld.sh

或者

bash ./helloworld.sh

或者

./helloworld.sh

这种方法执行脚本，必须有脚本的可执行权限。默认情况下，创建脚本是没有可执行权限的，此时可以使用 chmod 赋予权限：

chmod +x ./helloworld.sh

或者

source ./helloworld.sh

或者

. helloworld.sh

这里的.是一个命令。

3 变量

3.1 环境变量

分为系统环境变量和用户环境变量。常见的系统变量：$HOME、$PWD、$SHELL、$USER

或者分为全局环境变量和局部环境变量。局部变量只对当前的shell有效，全局变量对当前shell的子shell同样有效。

查看当前所有环境变量：

env

或者：

printenv

查看某个环境变量：

printenv HOME
